How to Use

upload_file

Uploading Files

Click the central drop zone or drag files from your file explorer. You can select multiple files at once. The tool supports images, documents, videos, audio files, and more.

edit

Configuring Rename Rules

Enter a base filename (e.g., "Project_Asset"). Optionally add a date/time stamp. Use the case conversion buttons to make all filenames uppercase or lowercase. The preview updates in real-time.

download

Downloading Results

Once you're satisfied with the preview, click "Download ZIP" to get a compressed archive containing all renamed files. The original files remain untouched on your system.

settings

Advanced Options

Filter files by type to work with specific categories. Sort files by name, size, or type. Drag rows in the preview table to manually reorder files before renaming.

security

Privacy & Security

This tool processes all files locally in your browser using JavaScript. No files are uploaded to any server. Your data stays completely private and secure. We do not collect any telemetry or usage data.
